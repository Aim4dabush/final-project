2. Start: Click on the clock symbol on the right to set start time of the event.

3. End: Click on the clock symbol on the right to set end time of the event.

4. Date: Click on calender symbol on the right to set the date of the event.

5. Address URL: Search the address in Google Maps. When Google finds the address copy the web link at the top and paste it in box.

6. Address: Simply enter the address of the event.
